# 博客介绍

- 前后端分离开发

> 默认是后台目录， 其中vue-views目录下是前端代码

# 启动项目
- node项目
```
node app.js
默认是3000端口
```
- vue项目
```
cd 进入 vue-views目录下
npm run dev
默认是8080端口
```

# 技术栈
- 前端
  + vue
  + vue-router
  + axios
  + element-ui

- 后端
  + koa2 + koa-router
  + mongoDB
